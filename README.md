# api-orchestrator-simple


Project Description

This project comprises three Kubernetes deployments:

API Service 1 (Flask):
Deploys a Flask-based API service handling specific functionalities.
Exposed externally through an Ingress controller.
API Service 2 (Flask):
Another Flask-based API service with distinct features.
Both API services are externally exposed using an Ingress controller.
MySQL Database:
Deploys a MySQL database to store and manage data.
